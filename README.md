# 🚀 Custom API

A RESTful API built with Django REST Framework that demonstrates CRUD operations, serializers, routing, and JSON-based API responses.

## ✨ Features

- Create, Read, Update and Delete APIs
- Django REST Framework
- JSON Responses
- URL Routing
- Serializer Validation
- Model-based APIs

## 🛠️ Tech Stack

- Python
- Django
- Django REST Framework
- SQL

## Installation

```bash
git clone https://github.com/Suma-s-h/customeapi.git

cd customeapi

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

## API Features

- GET
- POST
- PUT
- DELETE
